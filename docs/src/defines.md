# Definitions
Here you will find an assortment of definitions of words!

### CLI
A CLI is a terminal.

### void pointers
Void pointers are used to point to any type of information. From a string, to a list, to a 512-bit signed integer.

### SIMD
`SIMD` stands for Single Instruction Multiple Data.
It usually offers data processing past the natural channel size the CPU supports.

### O
In programming the complexity of an operation can be represented with `O()`.
Inside O is usually an `n`, along with some operation, or a one.
A *really* good example of O(1) complexity, is something like adding 1 and 1.
It is "trivial"
Generally, the more complex, the worse it is. Both in time, and work the CPU must do.
A good example of a complex `O`, is naive array multiplication. It is simple for humans, but it takes a lot for a CPU to do. Especially as you add more and more.
