2) SHELL COMMANDS
  show                 Clear the screen and list your buffer with line numbers
  run                  Compile `code` to bytecode and execute it
  save <file>          Save the buffer to <file>
  load <file>          Replace the buffer with <file> and write it to `code`
  replace OLD || NEW   On each line, replace the first OLD with NEW (mind the spaces and pipes)
  info                 Show line count and byte size of `code`
  clear mem [-y]       Clear buffer and truncate `code` (`-y` skips confirmation)
  cls                  Clear the terminal
  help                 Print this guide (reads `guide.txt`)
  dir                  Show absolute paths for `code`, `guide.txt`, and `pref`
  exit                 Quit Avocado

Editing basics:
  - N <text> creates/replaces line N (1-based).
  - Skipped numbers create blank lines. The editor persists to `code`.

3) MINIS LANGUAGE QUICK REF

3.1 Comments
  // line comment
  /* block comment */

3.2 Statements & Types
  - End each statement with a semicolon.
  - Types: int, float, bool, str, list, and void (for function returns).
  - `auto` lets the value decide the type.

  let int a = 1;
  let auto s = "hello";
  a = a + 1;

3.3 Expressions & Operators
  + - * /              arithmetic ( + concatenates strings too )
  == != < <= > >=      comparisons
  && ||                logical and / or
  ( ... )              grouping

3.4 Lists
  Literal: [1, 2, "a", true]
  0-based indexing
  Built-ins:
    len(x)                 // works for strings and lists
    ListGet(xs, i)
    ListSet(xs, i, v)
    ListPush(xs, v)

3.5 Control Flow
  if (cond) { ... } elif (cond) { ... } else { ... }
  while (cond) { ... }
  continue;               // next loop iteration
  break;                  // exit nearest loop
  break N;                // exit N nested loops (e.g., break 2;)

3.6 Functions (Typed Returns)
  // Void return: no return needed
  func void greet(str who) {
    print("Hello, " + who + "!\n");
  }

  // Value return: return a value of the declared type
  func int add(int a, int b) {
    return a + b;
  }

  // Legacy: func name(...) { ... } still works; return if you produce a value.

3.7 Misc
  exit;                   // stop the program immediately
  import "name";          // parsed, no-op (use shell `load` to open files)

6) EXAMPLES

Hello

Loop
  1 let int n = 3;
  2 while (n != 0) {
  3   print("tick\n");
  4   n = n - 1;
  5 }

Lists
  1 let list xs = [1, 2, 3];
  2 ListPush(xs, 7);
  3 print("len=" + len(xs) + "\n");
  4 print("first=" + ListGet(xs, 0) + "\n");

Raw key echo (ESC quits)
  1 let bool go = true;
  2 Input.Start();
  3 print("Press keys (ESC to quit)\\n");
  4 while (go) {
  5   let str k = Input.Key();
  6   if (k != "") {
  7     if (k == "\x1B") { go = false; }
  8     else { print("You pressed: " + k + "\n"); }
  9   }
 10 }
 11 Input.Stop();
